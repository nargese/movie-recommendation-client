Movie Recommendation — Frontend Client
A modern React web app that helps users discover movies and receive personalized recommendations based on their ratings and preferences.

Tech Stack
React (Vite)
JavaScript (ES6+)
Axios — API communication
React Router — Navigation
Bootstrap / CSS — UI styling

Features
🔐 User authentication (login / register)
🎥 Browse movie catalog
⭐ Rate movies
🤖 Recommendations
🔎 Search movies
📱 Responsive design
